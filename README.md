# ChatPDF

Ask anything to PDFs.
[Demo](https://chatpdfs.streamlit.app/)

## Installation

Developed using `python 3.10` on windows.

```bash
pip install -r requirements.txt
```
## Run
```bash
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```
## Usage

```bash
streamlit run app.py
```
